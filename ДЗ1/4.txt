 Создаем список нечетных значений от 1 до 60
odd_numbers = [num for num in range(1, 61) if num % 2 != 0]

# Выводим числа, делящиеся на 3 или на 5 и при этом не делящиеся на 15
filtered_numbers = [num for num in odd_numbers if (num % 3 == 0 or num % 5 == 0) and num % 15 != 0]

# Выводим отфильтрованные числа
print("Числа, делящиеся на 3 или на 5 и не делящиеся на 15:")
print(filtered_numbers)

# Выводим последний элемент созданного списка нечетных чисел
if odd_numbers:
    print("Последний элемент списка нечетных чисел:", odd_numbers[-1])

### Объяснение кода:

1. Создание списка нечетных чисел:
   - Генератор списка [num for num in range(1, 61) if num % 2 != 0] создает список odd_numbers, содержащий нечетные числа от 1 до 60.

2. Фильтрация чисел:
   - Второй генератор списка [num for num in odd_numbers if (num % 3 == 0 or num % 5 == 0) and num % 15 != 0] создает список filtered_numbers, который содержит числа, делящиеся на 3 или 5, но не делящиеся на 15.

3. Вывод результатов:
   - Сначала выводится список отфильтрованных чисел.
   - Затем проверяется, не пустой ли список odd_numbers, и выводится последний элемент с помощью odd_numbers[-1].
