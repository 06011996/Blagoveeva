Инициализация пустого списка для хранения чисел
numbers = []

# Ввод чисел от пользователя
while True:
    user_input = input("Введите число (-1 для завершения ввода): ")
    number = int(user_input)
    
    if number == -1:
        break  # Завершаем ввод, если введено -1
    
    numbers.append(number)  # Добавляем число в список

# Шаг 1: Вывод длины списка
length_of_list = len(numbers)
print(f"Длина списка: {length_of_list}")

# Шаг 2: Вывод суммы элементов списка через цикл
sum_via_loop = 0
for num in numbers:
    sum_via_loop += num
print(f"Сумма элементов списка (через цикл): {sum_via_loop}")

# Шаг 2: Вывод суммы элементов списка через метод списка
sum_via_method = sum(numbers)
print(f"Сумма элементов списка (через метод): {sum_via_method}")

# Шаг 3: Вывод только четных элементов списка
even_numbers = [num for num in numbers if num % 2 == 0]
print(f"Четные элементы списка: {even_numbers}")

### Объяснение кода:

1. Ввод чисел от пользователя:
   - Мы используем бесконечный цикл while True для ввода чисел. Если пользователь вводит -1, ввод прекращается с помощью break.
   - Каждое введенное число добавляется в список numbers.

2. Длина списка:
   - Используется встроенная функция len() для вычисления и вывода длины списка.

3. Сумма элементов списка:
   - Через цикл: Мы инициализируем переменную sum_via_loop и используем цикл for, чтобы суммировать элементы списка.
   - Через метод списка: Мы используем встроенную функцию sum() для получения суммы элементов списка.

4. Четные элементы списка:
   - Мы используем списковое включение (list comprehension) для создания нового списка, содержащего только четные числа.
